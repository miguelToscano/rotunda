## Getting set up

- git clone `https://github.com/miguelToscano/rotunda.git`
- cd rotunda/url-parser
- npm i
- make sure you have port 3000 available
- npm start

## Endpoints

- GET /urls/schema
- POST /urls/schema
- POST /urls/parse
